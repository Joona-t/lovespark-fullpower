# LoveSpark Full Power Extensions

Aggressive personal-use browser extensions. Not on the Chrome Web Store — sideload only.

| Extension | Purpose |
|-----------|---------|
| [lovespark-cookie-nuke](lovespark-cookie-nuke/) | Aggressive cookie consent banner removal |
| [lovespark-popup-fortress](lovespark-popup-fortress/) | Popup/redirect blocker with API overrides |
| [lovespark-yt-shield](lovespark-yt-shield/) | Three-layer YouTube ad neutralizer |

## Install

1. Clone this repo
2. Go to `chrome://extensions` and enable **Developer mode**
3. Click **Load unpacked** and select any extension folder

See each extension's README for full details and Firefox instructions.

## Privacy

All three extensions: zero data collection, zero analytics, zero outbound network calls. Everything stays local.
